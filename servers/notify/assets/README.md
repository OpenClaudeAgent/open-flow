# Assets

Place your notification icon here as `icon.png`.

## Requirements

- **Format**: PNG (recommended) or ICNS
- **Size**: 128x128 pixels minimum (256x256 recommended for Retina)
- **Name**: `icon.png`

## Usage

The notification system will automatically use this icon when sending notifications via PyObjC on macOS.

If no icon is found, notifications will be sent without a custom icon.
