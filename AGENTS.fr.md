# Instructions Globales

## Gestion des taches

**OBLIGATOIRE** : Pour toute tache non-triviale (3+ etapes), utilise l'outil TodoWrite pour :
1. Creer la liste des taches au debut
2. Marquer chaque tache `in_progress` quand tu la commences
3. Marquer chaque tache `completed` des qu'elle est terminee

L'utilisateur doit pouvoir suivre ta progression a tout moment.

## Exceptions

Ne pas utiliser les todos pour :
- Questions simples / reponses informatives
- Taches triviales (< 3 etapes)
- Corrections mineures rapides
